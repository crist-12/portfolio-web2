¿Qué puedo hacer?

Desarrollo web

Puedo crear webapps con el stack MERN (Mongo - Express - ReactJS - NodeJS), sin embargo estoy dispuesto a aprender otras tecnologías que puedan aplicarse al rubro. De igual manera, he utilizado AWS para hacer el control de la webapp en la nube y plataformas para hacer deploy como Heroku y Firebase.

Desarrollo móvil

En el área de desarrollo móvil me he desenvuelto en la librería de React Native bajo la plataforma Expo, utilizando manejo de estados con ContextAPI, manejo local de datos con SQLite y manejo en base de datos en la nube como Firebase Storage y Firestore, control de notificaciones con Expo Push Notifications y administración de usuarios con Firebase Auth.

Base de datos

En el ámbito de las bases de datos poseo conocimientos de administración y manejo de bases de datos relacionales (SQL, MySQL, MariaDB) y bases de datos no relacionales (MongoDB, Firebase Storage)
